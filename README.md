# Practicas sobre AJAX y jQuery

## AJAX Asynchronous JavaScript XML

CRUD operaciones llevadas sobre un DB

Crear un objeto de XMLHttpRequest para intercambiar datos con el servidor

```
var xhttp = new XMLHttpRequest();
```
